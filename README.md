# DataSceince

Designed a Convolutional Neural Network (CNN) to recognize individuals in video footage, with the ability to identify the same person across different videos. This project involved utilizing advanced deep-learning techniques for feature extraction and classification.

Create a data set from different videos using Python to extract frames.
Create subsets for training, validation, and testing.
Setup hyperparameters (number of epochs, batch size, input shape).
Setup training and validation data generators (used TensorFlow Keras)
Setup CNN model and trained Training and validation accuracy is 99% using matplot to visualize
Code with Generative AI. Evaluate the model on test data accuracy is 0.05% and accuracy is 98.6%
Use the trained model to predict the character that appears in a new video frame number range and the time in seconds.
